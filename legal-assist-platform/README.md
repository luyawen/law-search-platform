# 法律法规辅助平台（前端原型）

Vue 3 + Vite + Element Plus 构建的 B 端管理后台原型，数据均为前端 Mock，无需后端。

## 功能

- **法律法规知识库**：目录树分类、关键字/流程/预案检索、数据表格、拖拽导入、详情抽屉
- **标准流程工具**：流程逐项通过/不通过评定、LocalStorage 持久化、导出执行报告

## 启动

```bash
cd legal-assist-platform
npm install
npm run dev
```

浏览器访问终端提示的本地地址（通常为 http://localhost:5173）。

## 构建

```bash
npm run build
npm run preview
```
